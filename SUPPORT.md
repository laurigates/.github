# Support

## Getting Help

For questions about a specific project, open an issue in that repository using the appropriate template.

## Bug Reports

Use the **Bug Report** issue template in the relevant repository. Include reproduction steps, expected behavior, and environment details.

## Feature Requests

Use the **Feature Request** issue template. Describe the problem you're solving and the proposed solution.

## Security Issues

See [SECURITY.md](SECURITY.md) -- do not report security vulnerabilities as public issues.
